Launch Lakkhi Application

Step 1: Open a new terminal (not in VS Code) and run:

cd ~/projects/lakkhi-program/integrated-app/backend && source venv/bin/activate && python manage.py runserver

Step 2: Open another new terminal (not in VS Code) and run:

cd ~/projects/lakkhi-program/integrated-app/frontend && npm start

Step 3: Click the button below to open the app in your browser:

Open Lakkhi App

Important: Using separate terminals outside VS Code prevents the disconnection issues.

If you need to stop the servers, press Ctrl+C in each terminal window.
