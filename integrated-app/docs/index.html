Lakkhi Campaign Creation System - Technical Documentation

Overview

The enhanced campaign creation system provides a feature-rich interface for creating blockchain-funded campaigns with sophisticated content creation tools, template selection, and SEO optimization.

Core Components

1. ImageUploader

A versatile image upload component with multiple import methods:

  • Drag-and-drop file upload
  • Image URL import
  • Clipboard paste support
  • Cloud storage integration
  • Image preview with edit/remove capabilities

2. RichTextEditor

A full-featured text editor for campaign descriptions:

  • Formatting toolbar (bold, italic, lists, etc.)
  • Image and video embedding
  • AI content suggestions
  • HTML editing mode
  • Fullscreen mode
  • Content metrics (word/character count)

3. CampaignTemplateSelector

A template library for quick campaign creation:

  • 20+ industry-specific templates
  • Category filtering
  • Search functionality
  • Template preview with icons and descriptions
  • Pre-filled content for each campaign type

4. SeoOptimizer

SEO tool for campaign optimization:

  • Meta title and description editor
  • Social media preview (Facebook, Twitter)
  • Keyword density analysis
  • SEO score calculation
  • Optimization suggestions

Main Campaign Page Implementation

The CreateCampaignPage.js integrates all these components in a multi-tab interface:

  1. Basics Tab
    • Campaign title, category, tags
    • Funding goal with token validation
    • Blockchain selection
    • Campaign duration
    • Contract owner designation
    • Image upload with ImageUploader
  2. Story Tab
    • Project story using RichTextEditor
    • Project goals
    • Team information
    • Risks & challenges
  3. SEO & Visibility Tab
    • Meta information for search engines
    • Social media preview customization
    • Keyword optimization
  4. Preview Tab
    • Live preview of the campaign
    • Campaign summary
    • Form validation status

Blockchain Integration

Key blockchain features in the system:

  1. Token Validation
    • Supports multiple blockchains (Ethereum, BSC, Base)
    • Token verification with symbol and decimal retrieval
    • Automatic blockchain detection
  2. Network Switching
    • Detects network mismatches (when wallet network differs from token blockchain)
    • Provides network switching functionality
  3. Price Oracle Integration
    • Fetches token prices from multiple sources:
      • On-chain DEX queries
      • Backend API
      • CoinGecko
      • Binance API
    • Converts between USD/EUR/GBP and token amounts
  4. Smart Contract Deployment
    • Campaign factory interaction
    • Milestone definition on-chain
    • MetaMask transaction handling

Form Persistence and Validation

  • Draft saving with localStorage
  • Field validation with custom validators
  • Progress tracking across tabs
  • Error reporting with specific field errors

Implementation Examples

Image Uploader Usage

<ImageUploader
  value={formData.basics.projectImageUrl}
  onChange={(imageData) => {
    if (imageData) {
      handleInputChange('basics', 'projectImageUrl', imageData.preview);
      if (imageData.file) {
        setFormData(prev => ({
          ...prev,
          basics: {
            ...prev.basics,
            projectImageFile: imageData.file
          }
        }));
      }
    }
  }}
/>

Rich Text Editor Implementation

<FormField
  label="Project Story"
  as={RichTextEditor}
  value={formData.story.projectStory}
  onChange={(content) => handleInputChange('story', 'projectStory', content)}
  required
  placeholder="Describe your project in detail."
  validate={(val) => val && val.trim().length >= 100}
  errorMessage="Story must be at least 100 characters"
  height="300px"
/>

Template Selector Integration

<CampaignTemplateSelector
  show={showTemplateSelector}
  onHide={() => setShowTemplateSelector(false)}
  onSelect={handleTemplateSelect}
/>
